# 🎓 Student Authentication System

A modern **Student Authentication System** developed using **PHP** and **MySQL** as part of **Task-3** of my **Web Development Internship at ApexPlanet Software Pvt. Ltd.**

This project provides a secure authentication system along with complete student record management using CRUD operations. It also includes advanced features such as search functionality, pagination, session management, and an improved user interface.

# 🚀 Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into the **htdocs** directory.
4. Open phpMyAdmin.
5. Create a database named **student_auth**.
6. Create the required tables (**users** and **students**) or import the SQL file.
7. Open your browser and run:

```
http://localhost/student_authentication_system/register.php
```

8. Register a new account.
9. Login and manage student records.
